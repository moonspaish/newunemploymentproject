Explanation

This is a voting app that collects user input through a form, organizes the data using Django’s models and stores the votes in a PostgreSQL database hosted on Railway.

Storing geometry data using SQL seems pretty harsh, and since it never changes I decided to store it as a GeoJSON on Github. Once I had both the GeoJSON and the collected data I had to use the cleaned county names as the common key that makes it possible to illustrate the data.
