What inspired me to do this project

This is the source of it all, a wikipedia article I've stumbled upon while looking at past elections on wikipedia. The map in particular peaked my interest, I knew for certain it cannot be easy to create something like this, and so began quite the journey where with a goal in mind I've learned quite a lot.

After I've set my goal, the next step was attempting to find:

  • Data with the election results
  • Data with the boundries of the counties and UATs as polygons
  • Any similar projects I can learn from
  • Documentation for how to create the maps

The resource that helped me the most

Raluca Nicola's project about the results of the 2019 European Parliament results in Romania helped me tremendously, it showed me most of the steps required, taught me how to use the data, what needed to be done and in what order. I really owe a lot of thanks to Raluca for publishing her project.

The code, the data and the process

Everything is posted on github here. The entire project from github relies only on python and different libraries for handling and illustrating data and I think it is a useful resource for anyone who is looking into similar topics especially since my approach avoided GIS software, thus creating less friction for anyone interested.
